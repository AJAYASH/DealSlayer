Nowadays people are in search for better online deals and always searching different online portals. 
Searching these deals individually on different websites takes a lot of time and resources.
This effort is being made to develop an API or web tool which provides user a detailed comparison of a product from different websites at one spot.
Different web scrapping techniques can be used for developing the API. 
In this case, particularly we are using an implementation of “Python” and “Selenium” techniques.
